# React TODO App

## Dependencies

- React
- Express (Server-side rendering)

## Start server

```sh
yarn start
```

## Development

- Babel
- Webpack
- ESLint
- Postcss

Run on local browser:

```sh
yarn dev
```

## Test

- Enzyme
- Mocha
- Power Assert
- Sinon
- TestCafe

Run tests:

```sh
yarn test
```

In watch mode:

```sh
yarn test:watch
```

Run E2E tests:

```sh
yarn build
yarn start
yarn test:e2e
```
