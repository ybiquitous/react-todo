# React TODO App

## Dependencies

- React

## Development

- Babel
- Webpack
- ESLint
- Browsersync

Run on local browser:

```sh
$ npm run watch
```

## Test

- Enzyme
- Mocha
- Chai
- Sinon
- WebdriverIO

Run tests:

```sh
$ npm test
```

In watch mode:

```sh
$ npm test -- -w
```

Run E2E tests:

```sh
$ npm run build && npm start
```

```sh
$ npm run webdriver:start
```

```sh
$ npm run test:e2e
```
