ToDo web application as playground via [React](https://reactjs.org/).

This is an **Universal JavaScript** web application.

## Stack

- Runtime
  - React
  - Express
- Development
  - Webpack
  - Babel
  - ESLint
  - Flow
  - Postcss
- Test
  - Enzyme
  - Tape
  - Sinon
  - TestCafe

## Command

To see all commands, please execute `npm run` or `yarn run` command.

### Setup

```sh
yarn
```

### Start server

```sh
yarn start
```

### Start development server

```sh
yarn dev
```

### Compile source files

```sh
yarn build
```

### Run unit test

```sh
yarn test
# yarn test:coverage
# yarn test:watch
```

### Run End-to-End test

```sh
yarn test:e2e
```
